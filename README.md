# BakeBook Web App — React + Vite PWA

A web-based migration of the BakeBook iOS app.

**Live App**: https://recalljeffchang.github.io/BakeBook-Web/

## Features
- 🏠 Home — Browse and search recipes
- 📖 Recipes — Full recipe list with smart ratio scaling
- ⏱️ Timer — Circular baking timer with presets
- 📓 Journal — Baking log with profit tracking
- 🧂 Inventory — Ingredient stock management

## Tech Stack
- React 19 + Vite 8
- Vanilla CSS (mobile-first)
- localStorage persistence
- GitHub Actions auto-deploy

## Development
```bash
npm install
npm run dev
```
